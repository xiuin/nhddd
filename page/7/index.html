✔️ mysite.com/和mysite.com是相同的URL，您无需设置重定向。

您的意思可能是mysite.com/foo和mysite.com/foo/不具有相同的内容。

在Django中，通常可以通过使用URLfoo/实现，然后默认设置APPEND_SLASH = True会将/foo重定向到/foo/。

✔️ 区别在于，if x == True:检查x的真值。 除0以外的所有整数的真值均为true（在本例中为2）。

if x == True:,但是，将x与True的值进行比较，True的值是1。比较2 == 1会得出错误的值。
